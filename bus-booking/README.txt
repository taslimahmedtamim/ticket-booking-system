Bus Ticket Booking - Quick Deploy (InfinityFree)
1) Upload this whole folder to htdocs (or upload the ZIP and Extract).
2) Edit db.php and set: $host, $user, $pass, $db from your MySQL details in cPanel.
3) If you didn't import tables yet, open phpMyAdmin and import schema.sql.
4) Visit: https://YOUR-SUBDOMAIN/bus-booking/index.php
